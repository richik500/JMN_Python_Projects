# Loan Approval Prediction and Model Comparison Using Python
This folder contains a Python hands-on project focused on analyzing loan applicant data, preprocessing records, building multiple Machine Learning models, comparing their performance, and predicting loan approval outcomes.

### 1. Dataset
Contains raw dataset file:
- loan_approval.csv

### 2. Scripts
Contains Python project files:
- loan_approval_prediction.py

### 3. Reports
Contains model outputs, charts, confusion matrices, and insights.

## Files Included

### 1. loan_approval_prediction.py
Python script covering:
- Loading CSV dataset using Pandas
- Missing value treatment
- Label Encoding categorical variables
- Feature scaling using StandardScaler
- Train-test split
- K-Nearest Neighbors (KNN)
- Naive Bayes Classification
- Logistic Regression
- Decision Tree Classifier
- Artificial Neural Network (ANN)
- Accuracy comparison of models
- Classification Reports
- Visualization of model performance

### 2. loan_approval.csv
Loan applicant dataset containing:
- Applicant income
- Co-applicant income
- Loan amount
- Loan term
- Credit history
- Gender
- Marital status
- Education
- Self-employed status
- Property area
- Loan approval status
