# Diabetes Prediction and Model Comparison Using Python
This folder contains a Python hands-on project focused on cleaning medical data, preprocessing features, building multiple Machine Learning models, comparing their performance, and predicting diabetes outcomes.

### 1. Dataset
Contains raw dataset file:
- diabetes.csv

### 2. Scripts
Contains Python project files:
- diabetes_prediction.py

### 3. Reports
Contains model outputs, charts, confusion matrices, and insights.

## Files Included

### 1. diabetes_prediction.py
Python script covering:
- Loading CSV dataset using Pandas
- Handling medically invalid zero values
- Median imputation for missing values
- Exploratory Data Analysis (EDA)
- Correlation heatmap
- Feature scaling using StandardScaler
- Train-test split
- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Confusion Matrix comparison
- Accuracy and Classification Reports
- Cross Validation
- Bagging and Boosting techniques

### 2. diabetes.csv
Medical dataset containing:
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (0 = No Diabetes, 1 = Diabetes)
