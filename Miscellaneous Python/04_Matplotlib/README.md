# Matplotlib and Data Visualization Using Python
This folder contains a Python hands-on project focused on creating different statistical plots, visualizing relationships in data, and interpreting insights using Matplotlib, Seaborn, and Pandas.

### 1. Dataset
Contains raw dataset file:
- tips.csv

### 2. Scripts
Contains Python project files:
- different_plots.py

### 3. Reports
Contains generated charts, visual insights, and interpretation summaries.

## Files Included

### 1. different_plots.py
Python script covering:
- Loading CSV dataset using Pandas
- Pairplot visualization
- Correlation Heatmap
- Correlation matrix interpretation
- Boxplot analysis
- Violin plot comparison
- Split violin plot
- FacetGrid multi-category scatter plots
- Data storytelling through charts

### 2. tips.csv
Restaurant tipping dataset containing:
- Total bill amount
- Tip amount
- Gender
- Smoking status
- Day of visit
- Meal time
- Group size
