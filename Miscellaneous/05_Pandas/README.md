# Python Pandas Functions and Excel Data Analysis
This folder contains a beginner-to-intermediate Python project focused on using Pandas functions to load, inspect, analyze, and explore Excel datasets. The project demonstrates essential Pandas operations widely used in Data Analytics, Business Intelligence, and Machine Learning preprocessing workflows.

### 1. Dataset
Contains raw dataset file:
- lifeexpectancy-data-Data&Metadata.xls.xlsx

### 2. Scripts
Contains Python project files:
- Python_Pandas.py

### 3. Reports
Contains output summaries, charts, and exploratory findings.

## Files Included

### 1. Python_Pandas.py
Python script covering:
- Printing basic Python output
- Loading Excel files using Pandas
- Reading spreadsheet data into DataFrame
- Checking dataset type
- Shape analysis (rows and columns)
- Column name extraction
- Data type inspection
- Viewing first records using head()
- Viewing last records using tail()
- Row slicing using index ranges
- GroupBy aggregation
- Bar chart visualization
- Dataset structure exploration using info()

### 2. lifeexpectancy-data-Data&Metadata.xls.xlsx
Global life expectancy dataset containing demographic and country-level indicators.
Typical columns include:
- Country / Region
- Life Expectancy
- Income Group
- Continent
- Population Indicators
- Metadata variables
- Regional averages
