# CSV File Description:
The dataset contains bank marketing campaign data with customer demographic, financial, and communication details. It is commonly used to predict whether a client will subscribe to a term deposit after marketing contact.

# Pandas Data Loading and Exploratory Data Analysis
This folder contains a Python hands-on project using Pandas, Matplotlib, and Seaborn for loading and analyzing a real-world bank marketing dataset.

## Files Included

### 1. bank_marketing_eda.py
Python script covering:
- Importing Pandas
- Reading CSV files using `read_csv()`
- Handling semicolon separated data
- Viewing dataset structure
- DataFrame attributes
- Sorting values
- GroupBy analysis
- Basic Exploratory Data Analysis (EDA)

### 2. bank-additional-full.csv
Bank marketing campaign dataset containing:
- Customer age
- Job type
- Marital status
- Education
- Loan / Housing details
- Contact information
- Campaign outcomes
- Target variable (`y`) indicating subscription status
